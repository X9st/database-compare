# MySQL + 达梦上线 Go/No-Go 清单

## 1. 必过门禁（全部满足才 Go）

- [ ] TC01-TC12 全部执行且通过
- [ ] 无 P1/P2 未关闭缺陷
- [ ] 核心链路可用：建源、测连、拉表、比对、结果查看、导出

- [ ] 配置业务主键后 `cmp_no_pk_log` 可进入行级比对
