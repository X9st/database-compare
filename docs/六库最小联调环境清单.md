# 六库最小联调环境清单

## 通用要求
- 每种数据库准备独立测试库/Schema: `qa_compare_*`
- 账号需具备:
  - 连接权限
  - 元数据查询权限（表/字段/索引/约束）
  - 测试库读写权限（用于造数）
- 防火墙放通应用所在机器到数据库端口
- 客户端驱动安装齐全（见下方）

## MySQL
- 默认端口: `3306`
- 驱动: `pymysql`
- 建议权限: `SELECT, INSERT, UPDATE, DELETE, CREATE, DROP, INDEX`

## PostgreSQL
- 默认端口: `5432`
- 驱动: `psycopg2-binary`
- 建议权限: 对目标 schema 的 `USAGE + CREATE + DML`

## Oracle
- 默认端口: `1521`
- 驱动: `oracledb`（或 `cx_Oracle`）
- 需确认: `service_name/SID`、字符集、NLS 设置

## SQL Server
- 默认端口: `1433`
- 驱动: `pyodbc` + ODBC Driver 17/18
- 需确认: SQL 登录模式开启、TLS 策略可连接

## DM（达梦）
- 默认端口: `5236`
- 驱动: DM Python 驱动（按环境安装）
- 需确认: schema 与大小写敏感策略

## Inceptor
- 常见端口: `10000`
- 驱动: `pyhive` / `impyla`（按部署协议）
- 需确认: 鉴权方式（用户名密码/Kerberos）与 catalog/schema

## 联调矩阵（环境补齐后执行）
- 同构 6 组:
  - MySQL→MySQL
  - PostgreSQL→PostgreSQL
  - Oracle→Oracle
  - SQL Server→SQL Server
  - DM→DM
  - Inceptor→Inceptor
- 异构环形 6 组:
  - MySQL→PostgreSQL
  - PostgreSQL→Oracle
  - Oracle→SQL Server
  - SQL Server→DM
  - DM→Inceptor
  - Inceptor→MySQL
