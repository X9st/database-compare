# 环境基线说明

## 目标
- 固定运行与测试环境，避免 `DEBUG`、`ELECTRON_RUN_AS_NODE` 等环境变量导致非代码问题。

## 后端基线
- Python: 建议 `3.11+`
- 依赖安装: `pip install -r requirements.txt`
- 启动前清理冲突变量:
```bash
unset DEBUG
```
- 若需显式指定:
```bash
DEBUG=false python main.py
```
- 测试建议:
```bash
DEBUG=false pytest
```

## 前端基线
- Node.js: 建议 `20+`
- 依赖安装: `npm install`
- 类型检查:
```bash
npm run lint
```
- 构建:
```bash
npm run build
```

## Electron 基线
- 启动前必须清理 `ELECTRON_RUN_AS_NODE`。
- 已在 `dev:electron` 脚本中内置:
  - 清理变量: `unset ELECTRON_RUN_AS_NODE`
  - 启动前日志: 输出当前 `ELECTRON_RUN_AS_NODE` 值

## 端口约定
- 后端: `18765`
- 前端: `3000`
